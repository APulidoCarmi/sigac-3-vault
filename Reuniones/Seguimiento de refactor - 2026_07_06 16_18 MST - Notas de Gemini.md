### **Resumen**

Revisión de flujos operativos para centralizar la gestión documental y validar datos mediante una factura consolidada.

**Interfaz y carga documental**  
Se presentó la interfaz del sistema como fuente única de verdad documental. Los clientes podrán gestionar sus archivos mediante una dirección web pública habilitada para carga directa.

**Integración del flujo operativo**  
Se discutió la necesidad de vincular obligatoriamente los documentos cargados con el perfil operativo de cada cliente. Este mapeo asegurará la trazabilidad y eliminará errores registrados en sistemas anteriores.

**Validación mediante factura glosada**  
Se estableció el uso de la factura glosada como la fuente definitiva para consolidar y validar datos contra los documentos cargados. Este proceso identificará discrepancias automáticamente durante la operación.
